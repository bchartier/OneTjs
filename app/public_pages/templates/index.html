OneTjs

OneTjs est un serveur implémentant le standard de l'OGC Table Joining Service.

Les opérations de type Data Access en HTTP GET sont implémentées :

  • GetCapabilities
  • DescribeFrameworks
  • DescribeDatasets
  • DescribeData
  • GetData

Eléments non supportés (à l'heure actuelle) :

  • Requêtes HTTP POST
  • Requêtes SOAP
  • WSDL
  • Valeurs de types ordinal et nominal
  • Le multilinguisme
  • Le champ documentation des attributs
  • Pour la requête GetCapabilities : les paramètres language, AcceptFormats, Sections et updateSequence
  • Pour la requête DescribeFrameworks : le paramètre language
  • Pour la requête GetData : les paramètres language, LinkageKeys , FilterColumn, FilterValue, XSL, Aid

Autres limitations :

  • Les jeux de données stockés dans une base SQL ne sont pas encore supportés
