TJS simple server

TJS simple server est un prototype de serveur TJS implémentant le standard de l'OGC Georeferenced Table Joining Service.

Seules les opérations de type Data Access en HTTP GET sont implémentées :

  • GetCapabilities
  • DescribeFrameworks
  • DescribeDatasets
  • DescribeData
  • GetData

Eléments non supportés pour l'instant :

  • Requêtes HTTP POST
  • Requêtes SOAP
  • WSDL
  • Valeurs de types ordinal et nominal
  • Le multilinguisme
  • Le champ documentation des attributs
  • Les exceptions décrites dans les spécifications
  • Pour la requête GetCapabilities : les paramètres language, AcceptFormats, Sections et updateSequence
  • Pour la requête DescribeFramewroks : le paramètre language
  • Pour la requête GetData : les paramètres language, LinkageKeys , FilterColumn, FilterValue, XSL, Aid

Autres limitations :

  • Chaque jeu de données (dataset) n'est associé qu'à un seul référentiel (framework)
  • Les jeux de données stockés dans une base SQL ne sont pas encore supportés
